## 核心概念

### Prompt 1.0 vs Prompt 2.0

| 维度 | Prompt 1.0 | Prompt 2.0 |
|------|------------|------------|
| 输入 | "帮我写一个XX功能" | "基于以下模型，实现XX功能..." |
| 输出 | 随机、不可控 | 符合模型结构 |
| 可维护性 | 低 | 高 |
| 与AI关系 | 运气 | 协作 |
