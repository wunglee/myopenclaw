## Day 1 详细设计（示例）

### 主题：发现墙 - 体验"维护地狱"

**目标：** 让学员切身感受"AI生成容易，维护难"

**教学过程：**
1. **引入**（10分钟）：展示一个"完美"的AI生成代码
2. **任务**（20分钟）：要求修改一个小功能
3. **困境**（20分钟）：学员发现代码难以理解和修改
4. **反思**（10分钟）：讨论问题的根源

**核心洞察：**
> 你缺的不是提示词，是"模型思维"。
