## 核心概念

### 什么是模型？

> 模型是对现实的**简化抽象**，用于理解和沟通复杂系统。

### 为什么需要模型？

1. **理解** - 把复杂变简单
2. **沟通** - 与团队对齐认知
3. **决策** - 指导代码组织
4. **与AI协作** - 让AI理解你的意图
