# Day 3 - 画模型：MECE原则入门

> 把直觉变成结构图，用MECE原则画出严谨的领域模型

---

## 课程目标

- **认知目标**：理解MECE原则（不重复、不遗漏）
- **技能目标**：掌握用MECE原则分解复杂系统
- **行动目标**：画出一张符合MECE原则的OpenClaw模型图

---

## 课程结构（60分钟）

| 阶段 | 时长 | 内容 | 形式 |
|------|------|------|------|
| 回顾 | 5分钟 | Day 2的模型作业回顾 | 展示 |
| 引入 | 15分钟 | 什么是MECE原则 | 讲解+案例 |
| 方法 | 20分钟 | 如何用MECE画模型 | 示范 |
| 实践 | 15分钟 | 重构你的模型图 | 实操 |
| 总结 | 5分钟 | MECE的边界与局限 | 讨论 |

## 核心概念

### MECE原则

> **Mutually Exclusive, Collectively Exhaustive**
> 相互独立，完全穷尽
