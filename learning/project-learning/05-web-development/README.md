# 05. Web 开发基础

Express 与 Fastify 两大框架入门。

## 目录

| 文件 | 内容 |
|------|------|
| `05-01-express-basics.ipynb` | Express 基础、路由、中间件 |
| `05-02-fastify-basics.ipynb` | Fastify 基础、Schema验证、插件 |

## 学习目标

- 掌握 Express/Fastify 基础
- 理解路由、中间件、插件
- 能构建简单 REST API

## 练习项目

用 Fastify 构建一个用户管理 API，包含：
- 用户 CRUD
- 请求验证
- 错误处理
