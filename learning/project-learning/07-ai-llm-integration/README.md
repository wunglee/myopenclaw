# 07 AI/LLM 集成

与大型语言模型交互的核心技术。

## 目录

| 文件 | 内容 |
|------|------|
| `07-01-openai-api.ipynb` | OpenAI API 基础、流式响应、结构化输出 |
| `07-02-function-calling.ipynb` | 工具调用、ReAct Agent |
| `07-03-embeddings-rag.ipynb` | Embeddings、向量检索、RAG |
| `07-04-multi-agent.ipynb` | Multi-Agent 系统 |

- OpenAI API 调用
- Function Calling / 工具调用
