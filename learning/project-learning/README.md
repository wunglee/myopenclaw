## 学习原则

1. **以用促学** - 每个知识点都要能运行、能修改
2. **对比学习** - 用Java思维理解TypeScript的差异
3. **模型先行** - 先画模型图，再写代码
4. **费曼技巧** - 学完后要能教会别人
